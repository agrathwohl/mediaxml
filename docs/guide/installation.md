Install `mediaxml` with [`npm`](https://www.npmjs.com):

```sh
$ npm install mediaxml
```

Or add to `package.json`:

```json
{
  "dependencies": {
    "mediaxml": "^0.1.0"
  }
}
```

See the [Getting Started](#getting-started) guide for more information
on using the `mediaxml` module.
